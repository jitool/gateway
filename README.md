# gateway管理

#### 介绍
Api-gateway+可视化操作管理平台，使用redis topic进行gateway节点动态刷新，支持多种限流策略、参数验证、权限校验、灰度路由等
若依项目链接：https://gitee.com/y_project/RuoYi
#### 软件架构
基于spring cloud gateway框架做的功能扩展
